## Descripción del proyecto

Este proyecto consiste en una aplicación web de gestión de reservas de viajes entre usuarios. La idea principal ha sido desarrollar una plataforma funcional en la que los usuarios puedan registrarse, consultar viajes, realizar reservas y gestionar su perfil, mientras que el administrador puede controlar usuarios, viajes y reservas desde un panel propio.

He intentado enfocar el proyecto como una aplicación realista, aplicando tanto la parte técnica como el diseño de la interfaz, para que el resultado final no sea solo funcional, sino también claro y usable.

## Objetivos

- Aplicar los conocimientos trabajados durante el ciclo en un proyecto completo.
- Desarrollar una aplicación web con autenticación, roles y paneles diferenciados.
- Gestionar viajes y reservas mediante operaciones CRUD.
- Trabajar con base de datos relacional y validaciones en frontend y backend.
- Presentar una aplicación estructurada, funcional y preparada para una entrega final.

## Funcionalidades principales

- Registro, inicio y cierre de sesión.
- Recuperación de contraseña y reactivación de cuenta.
- Roles de usuario y administrador.
- Gestión de perfil.
- Creación, edición y cancelación de reservas.
- Historial de reservas finalizadas.
- Gestión de viajes desde administración.
- Publicación de viajes por parte de usuarios conductores.
- Panel de administración para usuarios, viajes y reservas.
- Exportación de datos en CSV y PDF.
- API REST básica.
